# CS 300 Portfolio – Data Structures & Algorithms

This repo contains my CS 300 portfolio artifacts, including
the runtime and memory analysis of data structures from Project
One and the working code to sort and print CS courses from Project Two.

In these projects, I solved problems related to efficiently storing,
accessing, and sorting data. I applied vectors, hash tables, and
binary search trees to analyze performance and memory usage.
Understanding data structures allowed me to choose the right tools
for each task and optimize my solutions. Roadblocks, such as handling
sorting and lookup efficiently, were overcome by testing different
structures and iterating on my code. These projects strengthened my
ability to design maintainable, readable, and adaptable programs, while
highlighting the importance of planning and analyzing software
performance before implementation.
